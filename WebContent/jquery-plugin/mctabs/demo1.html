Persistence

Setting {persist: true} in the McTabs script will turn on the persistence feature: the most recently clicked tab will be remembered even if the page is reloaded or revisited within the browser session.



Note:

If you open this page directly by local path instead of through HTTP on a web or localhost server, you may not see the persistence.

Multiple Tabs

You can have multiple McTabs on the same page, with only one copy of the CSS and JavaScript files.

Nested tabs are also supported.
